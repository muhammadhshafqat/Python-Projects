# Python Projects

Welcome to my Python Projects repository! Here, you'll discover a collection of Python projects designed to enrich your programming skills. Each project covers various aspects of Python development, from fundamental concepts to more advanced functionalities.

## Project 1: Calculator

### Description

The Calculator project is a versatile console-based calculator that I created to cater to a broad spectrum of mathematical needs. It's designed to be user-friendly and efficient, offering features suitable for both basic and advanced calculations.

### Features

1. **Basic Arithmetic Operations:**
   - Addition, Subtraction, Multiplication, Division
   - Exponentiation, Modulus

2. **Trigonometric Operations:**
   - Sine, Cosine, Tangent

3. **Mathematical Functions:**
   - Square Root, Logarithm, Factorial, Exponential Function

4. **Memory Operations:**
   - Memory Recall (MR), Memory Clear (MC)
   - Memory Addition (M+), Memory Subtraction (M-)

5. **Geometry Operations:**
   - Hypotenuse Calculation (hyp)

6. **Binary Operations:**
   - Bitwise AND (&), Bitwise OR (|), Bitwise XOR (^)

7. **Angle Unit Switching:**
   - Switch between degrees (deg) and radians (rad) for trigonometric functions.

8. **Efficient Error Handling:**
   - Handles invalid inputs gracefully, ensuring a smooth user experience.

9. **Memory Storage:**
   - Allows me to store results in memory with a user-defined key.

10. **User-Friendly Interface:**
    - Provides a simple and interactive interface for easy input and interaction.
